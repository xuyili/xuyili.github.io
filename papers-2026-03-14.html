📝 自回归（AR）视频生成模型依赖于将像素压缩为离散 token 序列的视频分词器（video tokenizer）。这些 token 序列的长度对于在重建质量和下游生成计算成本之间取得平衡至关重要。传统的视频分词器在不同的视频时间块上采用统一的 token 分配，常常在简单、静态或重复的片段上浪费 token，而在动态或复杂的片段上分配不足。为解决这种低效问题，我们提出 **EVATok**（**E**fficient **V**ideo **A**daptive **Tok**enizers）框架。我们的框架能够为每个视频估计最优的 token 分配，以实现最佳的质量‑成本权衡；开发轻量级路由器（router）用于快速预测这些最优分配；并训练自适应分词器，使其根据路由器预测的分配方案对视频进行编码。我们证明 EVATok 在视频重建和下游 AR 生成任务上实现了显著的效率提升和整体质量改进。得益于我们融合视频语义编码器的高级训练配方，EVATok 在 UCF‑101 数据集上实现了卓越的重建效果和领先的类‑视频生成性能，同时相较于先前最优的 LARP 以及固定长度的基线方法，平均 token 使用量至少节约 24.4%。

📝 多模态大型语言模型（MLLMs）正越来越多地被用于执行视觉工作流，例如导航图形用户界面（GUI），其中下一步操作取决于经过验证的视觉组合条件（例如“如果出现权限对话框且界面颜色为绿色，则点击允许”），并且该过程可能会分支或提前终止。然而，这一能力仍然缺乏充分评估：现有基准测试侧重于浅层组合或独立约束，而非深度链式组合条件。在本文中，我们引入了MM-CondChain，一个用于视觉 grounded 深度组合推理的基准测试。每个基准实例都被组织为多层推理链，其中每一层都包含一个基于视觉证据构建的非平凡组合条件，由多个对象、属性或关系组成。要正确回答，MLLM必须详细感知图像，在每一步对多个视觉元素进行推理，并遵循生成的执行路径直到最终结果。为了可扩展地构建此类工作流风格的数据，我们提出了一个智能体合成管道：规划器（Planner）负责逐层生成分组合条件，而可验证的程序化中间表示（VPIR）则确保每一层的条件在机械上是可验证的。随后，组合器（Composer）将这些经过验证的层组装成完整的指令。使用该管道，我们在三个视觉领域中构建了基准测试：自然图像、数据图表和GUI。

📝 我们推出 $Ψ_0$（Psi-Zero），一个用于解决挑战性人形机器人locomotion-manipulation任务的开源基础模型。现有的方法通常尝试通过在大型且多样化的人类和人形机器人数据上进行联合训练来解决这一基本问题，但我们认为由于人类和人形机器人之间存在根本性的运动学和运动差异，这种策略并非最优的。因此，尽管数据量相当可观，数据效率和模型性能仍然不尽如人意。为应对这一挑战，\ours\;将学习过程解耦，以最大化异构数据源的效用。具体而言，我们提出了一种具有不同学习目标的多阶段训练范式：首先，我们在大型自我中心人类视频上自回归地预训练一个视觉语言模型（VLM）主干，以获得可泛化的视觉-动作表示。然后，我们在高质量的人形机器人数据上后训练一个基于流（flow）的动作专家，以学习精确的机器人关节控制。我们的研究进一步发现了一个关键但经常被忽视的数据配方：与那些使用嘈杂的互联网片段或异构跨本体机器人数据集进行扩展的方法不同，我们证明在高质量的自我中心人类manipulation数据上进行预训练，然后在特定领域的真实世界人形机器人轨迹上进行后训练，可以获得卓越的性能。大量真实世界实验表明，\ours\;仅使用约800小时的人类视频数据和30小时的人形机器人数据就实现了最佳性能。
